# Chatter

Chatter is a real-time group chat application where users can join with unique usernames, see active status, and chat instantly. It is built with Node.js, Express, and Socket.io.

---

## Setting up Chatter to run Locally

### 1. Clone the Repository (do this if you dont have the project installed yet)

```bash
git clone https://github.com/KevSun15/Chatter.git
cd Chatter
```

### 2. Install dependencies (start here if you already have the project installed)

Make sure you have node.js installed and you are in the root folder of Chatter

```bash
npm install
```

### 3. Run Locally

```bash
npm run start
```

You should see 
```bash
Server running on http://localhost:3000
```
Go to http://localhost:3000 to see Chatter
